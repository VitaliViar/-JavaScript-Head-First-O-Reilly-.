# Как создаются команды
```javascript
<h1 class="drink">Mocha Caffe Latte</h1> /*При работе с HTML мы размечаем текст для определения структуры: например, «Так, здесь у нас будет крупный заголовок, а за ним следует абзац обычного текста».*/
<p>Espresso, steamed milk and chocolate syrup, just the way you like it.</p>

h1.drink { /*Для CSS пишутся правила с селекторами (например, h1.drink и p), определяющими, к каким частям разметки HTML применяется данный стиль.*/
color: brown; /*Допустим, все заголовки drink выводятся коричневым цветом...*/
}
p {
font-family: sans-serif; /*...А абзацы выводятся шрифтом без засечек.*/
}
```
### Набор команд
```javascript
var age = 25; /*Каждая команда выполняет небольшую часть работы, например объявляет переменные, в которых будут храниться используемые значения.*/
var name = "Owen"; /*Мы создаем переменную для хранения возраста (25). Нам понадобится
и переменная для имени “Owen”.*/
if (age > 14) { /*Значение переменной может использоваться для принятия решений. Возраст пользователя больше 14?*/
alert("Sorry this page is for kids only!"); /*Если больше — сообщаем, что пользователь слишком стар для этой страницы.*/
} else {
alert("Welcome " + name + "!"); /*А если нет — приветствуем пользователя по имени (впрочем, в нашем примере Оуэну 25 лет, так что сообщение не выводится).*/
}
```
