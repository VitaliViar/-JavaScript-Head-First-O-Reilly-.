# Поаккуратнее с выражениями!
```javascript
var total = price - (price * (discount / 100)); /*Команда JavaScript присваивает вычисленный, Знак * используется для умножения, а / — для деления. Наша
переменная total. В результате вычисления цена (price) уменьшается на размер скидки (discount), заданный в процентах от цены. Если переменная price содержит значение 10, а discount — 20, результат будет равен 8.*/
"Dear " + "Reader" + "," /*Эти строки «сцепляются» в новую строку «Dear
Reader» (такая операция называется конкатенацией).*/
"super" + "cali" + youKnowTheRest /*То же самое, но в выражение входит переменная, содержащая строку. При вычислении результата получается строка “supercalifragilisticexpialidocious”*/
phoneNumber.substring(0,3)/*Еще один пример выражения, возвращающего строку. О том, как оно работает, будет рассказано позднее, а пока достаточно сказать, что выражение возвращает код зоны из телефонного номера.*/
age < 14 /*Если возраст (age) меньше 14, то результат равен true; в противном
случае он равен false. Команда проверяет возраст пользователя.*/
cost >= 3.99 /*Если цена товара больше или равна 3.99, выражение истинно. В противном случае оно ложно. Не пропустите распродажу!*/
animal == "bear" /*Истинно, если в названии животного
присутствует слово “bear”.  *
Предполагается, что переменная youKnowTheRest содержит строку “fragilisticexpialidocious”.*/
```
